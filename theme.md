Create a modern, minimal, and professional theme for a web platform named "OmniPath".

The theme must represent a smart, inclusive, and privacy-first urban transit ecosystem. The visual identity should reflect reliability, accessibility, sustainability, and advanced AI-driven intelligence.

Theme Requirements:

- Style: Clean, futuristic, transport-tech inspired
- Tone: Trustworthy, inclusive, and efficient
- Color Palette:
  - Primary: Deep Blue (trust, reliability)
  - Secondary: Teal/Green (sustainability & mobility)
  - Accent: Soft Orange or Yellow (alerts, live updates)
  - Background: Light Neutral or Soft Gradient
- Typography:
  - Use clean sans-serif fonts
  - Clear hierarchy with readable body text
  - Accessibility-friendly contrast
- UI Mood:
  - Spacious layout
  - Soft shadows and subtle gradients
  - Rounded UI elements
  - Smooth transitions and micro-interactions

Brand Personality:

- Privacy-focused
- Real-time intelligent assistant
- Public transport convenience platform
- Inclusive for differently abled users

Ensure the theme supports:

- Dark mode and Light mode
- Mobile-first responsive design
- Offline-first design feel
